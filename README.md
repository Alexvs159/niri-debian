# niri-debian
Niri and xwayland-satellite package for Debian 13 Trixie. Built from [https://launchpad.net/~avengemedia/+archive/ubuntu/danklinux](https://launchpad.net/~avengemedia/+archive/ubuntu/danklinux) with Debian SDK tools (mk-buld-deps, dpkg-buildpackage).

# Installation
- Clone this repository:
```
git clone https://github.com/Alexvs159/niri-debian.git
```

- Install the packages with this command as normal user:
```
sudo apt install niri-debian/*.deb
```

- Enjoy
